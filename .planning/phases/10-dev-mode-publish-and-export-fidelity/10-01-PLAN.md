  <action>
    Create `packages/editor-core/src/dev-mode.ts`. Export:
    - `createDevModeInspectReport(bundle, input)`
    - `createDevCodeSnippet(bundle, input)`
    - `markReadyForDev(bundle, input)`
    - `markReadyForDevChanged(bundle, objectId, createdAt?)`
    - `createVersionDiffRecord(bundle, input)`
    - `createAssetDownloadRecord(bundle, input)`
    - append helpers for reports, snippets, diffs, and asset downloads.

    Dev Mode behavior:
    - Call `ensureCanvasGraph` before reading objects.
    - Unknown object throws `Unknown Dev Mode object: ${objectId}`.
    - Inspect reports derive node id, bounds, spacing, layout, CSS properties, tokens, component metadata, prototype metadata, and asset ids from stored bundle state.
    - Missing rendered rect emits accessibility note `missing_rendered_rect`.
    - Code snippets support `css`, `tailwind`, `reactProps`, and `tokenReference`.
    - Asset download requires an existing stable `ProjectAssetUrl`; otherwise throw `Asset download requires stable project URL: ${assetId}`.

    Create `packages/editor-core/src/export-fidelity.ts`. Export:
    - `createExportArtifactRecord(bundle, input)`
    - `createExportVerification(input)`
    - `createPublishPreview(bundle, input)`
    - `appendExportArtifact(bundle, artifact, verification?)`
    - `appendPublishPreview(bundle, preview)`

    Export/publish behavior:
    - `createExportArtifactRecord` rejects `sourceRevision !== bundle.baseRevision`.
    - `createPublishPreview` creates `url: "kdesign://publish/{projectId}/{publishId}"` and never uses `https://`.
    - `appendExportArtifact` requires the referenced `ExportJob`.
    - `appendPublishPreview` requires all artifact ids to exist.

    Create `packages/editor-core/src/code-roundtrip.ts`. Export:
    - `createCodeRoundtripPackage(bundle, input)`
    - `validateCodeRoundtripImport(bundle, input)`
    - `appendCodeRoundtripPackage(bundle, pkg)`
    - `appendCodeRoundtripImport(bundle, result)`

    Roundtrip behavior:
    - `manifestJson` contains serialized `ProjectBundle`, `baseRevision`, `canvasGraph`, `editGraph`, `assets`, `designSystem`, `sourceRecords`, `exportArtifacts`, and `instructionsPath`.
    - Validate runtime, source revision, object ids, node ids, allowed patch ops, allowed canvas operations, and unsafe patch values.
    - Store conflicts as diagnostics; do not apply patches in this helper.

    Add exports to `packages/editor-core/src/index.ts`.

    Add unit tests covering Dev Mode inspect/snippets/readiness/diff/assets, export artifact and publish records, and code roundtrip runtime/source/operation validation.
  </action>
